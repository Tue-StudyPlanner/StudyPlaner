# Agent Profiles

Use these profiles when a task needs specialized expertise.

- `system-architect.md` — architecture, scalability, boundaries, technical strategy
- `backend-architect.md` — APIs, databases, reliability, data integrity
- `frontend-architect.md` — UI, accessibility, responsive design, frontend architecture
- `security-engineer.md` — security audits, auth, compliance, threat modeling
- `performance-engineer.md` — profiling, bottlenecks, Core Web Vitals, optimization
- `deep-research-agent.md` — research, source validation, complex synthesis

Default rule: use the smallest suitable profile. Do not involve multiple profiles unless the task truly needs it.
