# StudyOS Frontend
This repository contains the React frontend of our application.  

## Prerequisites
Before you start, make sure the following is installed:

| Tool | Minimum version | Download |
|------|----------------|----------|
| **Node.js** | v20.12 or higher | [nodejs.org](https://nodejs.org) → download LTS |

> ⚠️ **Important:** Versions below v20.12 cause a `SyntaxError: styleText` error when creating the project. If this happens, download the current LTS version from nodejs.org.

## Run the project
To run the project, the following steps must be taken:

1. Navigate to the frontend folder
2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. The app is now running at: **http://localhost:5173**
